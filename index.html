How to improve accessibility?

A good start: Improve your HTML that your JavaScript renders

First: Improve your HTML/CSS Stuff

  • Ensure all form inputs have labels
  • 📸 Ensure there are alt attributes on your images with helpful text (if not decorative)
  • 🙅🏼 Don't have empty links or buttons
  • 🎨 Have accessible contrast
  • Have clear focus styling

Now let's get to JavaScript

Modals

  • Properly shifting focus to the modal on open
  • "Focus Trapping" - Ensuring that the focus stays in that modal unless you exit
  • Upon close, focus should be moved back to where it was before it moved into the modal.
  • Should close on ESC keypress
http://bit.ly/react-accessible-modal

Accordions

  • Using semantic buttons so click events have built in keyboard handling
  • Ensuring to keep all the sections expanded until JavaScript loads.
  • Then add ARIA expanded and hidden state in addition to styling.
http://bit.ly/vanilla-js-accordion

Overall Focus Management

  • Making sure the focus isn't going to visually hidden elements.
  • Making sure to properly navigate keyboard users where they should go.
http://bit.ly/js-a11y-tabs

Progressive Enhancement

  • Adding a class that gets removed first when JavaScript loads
  • Adding alternative styling if the JavaScript doesn't load
  • Adding aria attributes via JavaScript, especially if it indicates interactivity
http://bit.ly/accordion-progressive-enhancement

Creating integration tests

  • Using jest-axe to get automated testing integrated with your integration tests.
  • Testing proper heading structure
  • Testing focus management and interaction.
  • Ensuring IDs match properly with aria attributes.
http://bit.ly/testing-a11y-react-testing
